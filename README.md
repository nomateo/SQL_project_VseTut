# SQL_project_VseTut

# Витрина данных маркетплейса 'ВсёТут'

SQL-проект: разработка витрины данных и решение ad hoc задач 
на основе транзакционных данных e-commerce платформы.

## Что сделано
- Построена витрина данных на уровне user_id + region (CTE, множественные JOIN)
- Исправлена шкала рейтингов в исходных данных
- Решены 4 аналитические задачи: сегментация, ранжирование, 
  региональная статистика, анализ активности по месяцам

## Стек
PostgreSQL + CTE + Window Functions + CASE WHEN + DATE functions

## Данные
Закрытая учебная БД курса (ds_ecom schema).
